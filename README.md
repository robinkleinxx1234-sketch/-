# Spotify Song Skipper
Let you skip Spotify Songs
🎵 Spotify Song Skipper v1.0
The first official release of the Spotify Song Skipper! This lightweight utility runs in your system tray and allows you to skip Spotify tracks using custom global hotkeys.

✨ Features
Global Hotkeys: Skip songs even while gaming or working in other apps.
Easy Configuration: Change your hotkeys directly via the system tray menu (no config files needed!).
Background Mode: Runs silently without a console window.
Privacy Focused: No data collection, runs locally on your machine.
🚀 How to Use
Download SpotifySkipper.exe below.
Run the application (Administrator rights required for global hooks).
Right-click the tray icon to set your custom "Next" and "Previous" keys.
Enjoy seamless music control!
